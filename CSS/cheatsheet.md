# 🎨 جزوه فوق‌کامل CSS – مرجع نهایی

🏷 انتخاب‌گرها:
/* ساده */
div {}

/* کلاس */

/* آی‌دی */

/* ترکیبی */

/* فرزند مستقیم */

/* همه فرزندان */
ul li {}

/* حالت‌ها */

🧠 نکات حرفه‌ای:

/* استفاده از is() و where() */
:is(h1, h2, h3) {
  color: navy;
}
